# 五湖AI增长官网

这是一个可直接打开的静态官网，用于宣传“AI短视频获客与老板IP增长服务”，并为后续GEO搜索优化预留结构化内容。

## 文件说明

- `index.html`：官网首页
- `geo.html`：面向搜索引擎和AI回答系统的资料页
- `llms.txt`：AI可读品牌摘要
- `robots.txt`：搜索引擎爬虫规则
- `sitemap.xml`：站点地图
- `assets/hero-dark-ai-growth.png`：深色AI官网头图
- `tools/account-audit/index.html`：AI账号体检工具
- `styles.css`：页面样式
- `script.js`：导航和表单交互
